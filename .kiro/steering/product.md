# Debate Platform

A real-time structured debate platform where users can engage in formal debates with a 3-round format (opening, rebuttal, closing). The platform features:

- Turn-based debate system with support/oppose sides
- Audience voting and stance tracking (pre/post debate positions)
- Market-style pricing based on audience sentiment
- Reputation and prediction accuracy scoring
- Sandbox mode for new users (5 debates required before full voting weight)
- Threaded comments on debates
- Reaction system for arguments (agree, strong_reasoning)

The platform uses Neon Auth (Better Auth) for authentication, with extended user profiles stored locally.
